# Personal Portfolio Website

A clean and minimal portfolio website that showcases my programming projects and work in software development.

---

## Overview

This portfolio website provides a simple way to explore projects I have built.  
It includes a homepage introduction and a projects page that lists selected work with descriptions and links to the source code.

The design focuses on **simplicity, readability, and fast loading**.

---

## Features

- Clean and minimal design
- Project showcase page
- Direct links to GitHub repositories
- Simple navigation
- Responsive layout

---

## Projects

### Asteroids
A Space Invaders–style arcade game built in **Python using Pygame** where the player controls a spaceship, shoots incoming enemies, and avoids attacks while trying to survive and score points.

### Porsche Data Dashboard
A **data analysis dashboard** that collects and visualizes Porsche vehicle data using charts, statistics, and interactive dashboards to generate insights.

### Expense Tracker
A **budgeting application** that allows users to record expenses, track spending, and view financial statistics using charts and summaries.
